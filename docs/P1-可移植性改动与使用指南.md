# HAJIMI B 端可移植性改动说明（P0/P1）与使用指南

> 适用对象：新 clone 的组员、联调负责人  
> 最后更新：2026-07  
> P0 解决「跑不起来」；P1 解决「能开但像坏了」。P2/P3 为脚本/配置/CI 维护，**不改变 UI 布局与主题**。

---

## 一、改了什么？

### P0 — 让别人能跑起来

| 项 | 文件 | 说明 |
|----|------|------|
| 启动脚本去硬编码 | `scripts/start_client.bat`、`start_omniparser.bat`、`resolve_omni_root.bat` | 不再写死 `E:\...`；优先环境变量 → `where python` |
| 三级运行模式 | `README.md`、`docs/B端-组员快速启动.md` | UI 壳 / 本地联调 / 校园 GPU |
| 依赖自检 | `requirements.txt`、`scripts/check_ui_env.py` | 启动前检查 PyQt5、QtSvg、cv2、PIL、mss |
| 端口与 server 统一 | `config.py`、`server/` | 默认 A 端 **8010**；权威 A 端目录为 `server/` |

### P1 — 减少「能开但像坏了」

| 项 | 文件 | 说明 |
|----|------|------|
| 启动 health 解耦 | `ui/main_widget.py`、`core/api_client.py` | Mock 模式跳过 health 轮询；A 端未连时用中性 `system` 提示，非满屏 danger |
| 内网不可达提示 | `core/deployment_resolver.py`、`core/api_client.py` | 提示检查 VPN/隧道，或切换「本地启动」/ `HAJIMI_MOCK_ONLY=1` |
| 托盘/窗口图标 | `ui/main_widget.py` `_setup_tray()` | `svg_icon("logo")`；消除 `No Icon set` 警告 |
| 项目结构文档 | `docs/项目结构.md` | Native UI 为主路径 |
| A 端契约 | `core/api_client.py`、`core/env_sync.py`、`server/routes/demo.py` | redline、宽松 success；env 写 `LLM_*` / `OMNIPARSER_URL` |

## 二、移植后如何使用

### 模式 A：只看 UI（约 5 分钟）

```powershell
scripts\setup.bat
# 或手动：
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python scripts/check_ui_env.py

set HAJIMI_MOCK_ONLY=1
python main.py
```

**预期**：窗口弹出；一条灰色 system 提示「UI 演示模式…」；托盘有 logo。

### 模式 B：本地完整联调

```powershell
copy server\.env.example server\.env
# 编辑 LLM_API_KEY、OMNIPARSER_URL=http://127.0.0.1:8002
scripts\setup_server_env.bat
scripts\start_all.bat
```

| 服务 | 默认端口 |
|------|----------|
| OmniParser | 8002 |
| A 端 | 8010 |
| B 端 UI | `python main.py`（不设 Mock） |

验收：`python scripts\verify_integration.py`

### 模式 C：校园 GPU

1. VPN + SSH 隧道（见 `docs/校园GPU-B端联调清单_v2.md`）  
2. 系统设置 → 内网 API → `http://127.0.0.1:8010`  
3. `python main.py`

## 三、环境变量速查

| 变量 | 默认 | 用途 |
|------|------|------|
| `HAJIMI_MOCK_ONLY=1` | — | 纯 UI，不连 A 端 |
| `HAJIMI_PORT` | `8010` | A 端端口 |
| `HAJIMI_API_URL` | `http://127.0.0.1:8010` | B 端连 A 端 |
| `OMNI_ROOT` | 项目 `OmniParser/` | OmniParser 目录 |
| `VIDEO_RAG_PY` / `OMNI_PY` | 可选 | 指定 Python |

用户设置：`%LOCALAPPDATA%\HAJIMI\user_settings.json`（默认 `deployment_mode=local`，`ui_theme=current`）。

## 四、常见问题

| 现象 | 处理 |
|------|------|
| `check_ui_env.py` FAIL | `pip install -r requirements.txt` |
| 满屏红色 A 端错误 | `set HAJIMI_MOCK_ONLY=1` 或 `start_server.bat` |
| 内网不可达 | 检查隧道，或设置页改「本地启动」 |
| bat 找不到 Python | 用 `.venv\Scripts\python main.py` |
| 图标空白但窗口能开 | 缺 QtSvg；`pip install --force-reinstall PyQt5` |
