# HAJIMI 项目结构（B 端 Native UI）

> 最后更新：2026-07。主路径为 **PyQt5 Native UI**，非 Web 内嵌。历史 `server2/` 归档已移除，A 端请以 `server/` 为准。

```
HAJIMI_UI/
├── main.py                 # B 端入口
├── config.py               # B 端全局配置（端口、Mock 开关等）
├── core/defaults.py        # A/Omni 默认 URL/端口（单源）
├── requirements.txt        # B 端 UI 依赖
├── requirements-dev.txt    # GPU 远程脚本（paramiko）
├── assets/fonts/           # Luxury 主题字体（可选）
├── core/                   # 业务逻辑：API 客户端、截图、服务管理
├── ui/
│   ├── main_widget.py      # 主窗口（Native / Web 双路径，默认 Native）
│   ├── chat_bubble.py      # 聊天气泡
│   ├── overlay_anno.py     # 全屏标注层
│   ├── native/             # Native UI 实现
│   │   ├── medium_panel.py # 中等面板
│   │   ├── compact_bar.py  # 紧凑胶囊
│   │   ├── theme_manager.py + themes/  # QSS 主题包
│   │   └── layout_tokens.py            # 布局尺寸 token
│   └── web/                # 浏览器 Demo 参考（非生产默认路径）
├── server/                 # A 端 FastAPI（联调时启动）
└── scripts/                # 启动与健康检查脚本
    ├── setup.bat           # 一键 venv + pip + check_ui_env
    ├── start_ui.bat        # 仅 B 端 UI（轻量 python main.py）
    ├── start_all.bat       # OmniParser + A + B 完整栈
    └── check_ui_env.py     # 依赖自检
```
